Task1 Condition

You should check input value. It should be email checking. The algorithm : value should contain @. And @ cannot be first or last digit. Write function that has 1 argument - value of input. For example - "@alal" - should return false, "qwe@" - should return false, asd@@asd - should return false. "asd@asd" should return true
